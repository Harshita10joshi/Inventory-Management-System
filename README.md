# Inventory Management System

## Overview
This project is a simple Inventory Management System developed using Advanced Java concepts in Eclipse IDE. The application provides a graphical user interface (GUI) using Java Swing for managing inventory items efficiently.

The system allows users to add, update, delete, save, and load inventory records. Data is stored using CSV file handling in Java.

---

## Features
- Add new inventory items
- Update existing items
- Delete inventory items
- Display inventory data in table format
- Save inventory data to CSV file
- Load inventory data from CSV file
- User-friendly Java Swing GUI

---

## Technologies Used
- Java
- Advanced Java
- Java Swing
- File Handling
- Eclipse IDE

---

## Project Structure
- GUI developed using Java Swing
- Inventory data managed using ArrayList
- Data persistence implemented using CSV file handling

---

## Functionalities
### Add Item
Users can add new inventory items by entering:
- Item ID
- Item Name
- Quantity
- Price

### Update Item
Selected inventory records can be updated easily.

### Delete Item
Users can remove selected inventory items.

### Save Data
Inventory records are saved into a CSV file using FileWriter and PrintWriter.

### Load Data
Previously saved inventory data can be loaded from the CSV file using BufferedReader.
